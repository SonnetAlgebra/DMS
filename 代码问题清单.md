## 🟡 中优先级（建议修复）

### 问题 3：数据库连接未使用连接池 ✅ 已修复

**位置**：所有路由文件（`upload.py`, `data.py`, `detect.py` 等）

**问题代码**：
```python
conn = sqlite3.connect(DB_PATH)  # 每次请求新建连接
```

**影响**：高并发时连接开销大，性能下降

**解决方案**：
```python
# 在 database.py 中创建连接池
import sqlite3
from contextlib import contextmanager

DB_PATH = "dms.db"

@contextmanager
def get_db():
    conn = sqlite3.connect(DB_PATH)
    try:
        yield conn
    finally:
        conn.close()

# 路由中使用
from app.database import get_db

@router.post("/upload")
async def upload_csv(file: UploadFile = File(...)):
    with get_db() as conn:
        cursor = conn.cursor()
        # ... 操作数据库
```

### 问题 4：数据库连接异常时未关闭 ✅ 已修复（合并至 FIX-03）

**位置**：`backend/app/routers/detect.py`

**问题代码**：
```python
conn = sqlite3.connect(DB_PATH)
cursor = conn.cursor()
# ... 操作
if metric is None:
    conn.close()  # 正常关闭
    raise HTTPException(...)
# 但如果后面抛出异常，连接未关闭
```

**影响**：异常发生时连接泄漏，长期运行后资源耗尽

**解决方案**：
```python
conn = sqlite3.connect(DB_PATH)
try:
    cursor = conn.cursor()
    # ... 所有数据库操作
    conn.commit()
finally:
    conn.close()  # 确保始终关闭
```

### 问题 5：重复查询影响性能 ✅ 已完成（保持现状）

**位置**：`backend/app/routers/data.py`

**问题代码**：
```python
# 先查总数
cursor.execute("SELECT COUNT(*) FROM time_series WHERE metric_id = ?", (metric_id,))
total = cursor.fetchone()[0]

# 再查数据
cursor.execute("SELECT timestamp, value FROM time_series...")
```

**影响**：两次数据库往返，性能浪费

**解决方案**：
```python
# 使用窗口函数一次查询
cursor.execute("""
    SELECT timestamp, value, COUNT(*) OVER() as total
    FROM time_series
    WHERE metric_id = ?
    ORDER BY timestamp
    LIMIT ? OFFSET ?
""", (metric_id, limit, offset))

rows = cursor.fetchall()
total = rows[0][2] if rows else 0
```

### 问题 6：文件类型未验证 ✅ 已修复

**位置**：`backend/app/routers/upload.py`

**问题代码**：
```python
file: UploadFile = File(...)  # 接受任意类型文件
```

**影响**：可能上传非 CSV 文件导致解析失败

**解决方案**：
```python
from fastapi import UploadFile, File, HTTPException

async def upload_csv(file: UploadFile = File(...)):
    # 验证文件扩展名
    if not file.filename or not file.filename.endswith('.csv'):
        raise HTTPException(status_code=400, detail="仅支持 CSV 文件")
    
    # 验证 MIME 类型
    if file.content_type not in ['text/csv', 'application/vnd.ms-excel']:
        raise HTTPException(status_code=400, detail="文件类型不正确")
```

### 问题 8：重复代码未提取

**位置**：`detect.py` 和 `data.py`

**问题**：相同的统计量计算逻辑重复

**解决方案**：
```python
# services/statistics.py
def calculate_statistics(values: list[float]) -> tuple[float, float]:
    """计算均值和标准差"""
    if len(values) < 2:
        raise ValueError("数据点不足")
    mean = sum(values) / len(values)
    std = (sum((x - mean) ** 2 for x in values) / len(values)) ** 0.5
    return mean, std
```
